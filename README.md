# LUPACORE OS: A Revolutionary AI-Driven Operating System

## Project Goal
To develop a secure, proprietary, cloud-native operating system (LUPACORE OS) with an integrated AI core powered by the Gemini model. 

## Phase 1: Introductory Site
Create a single-page website introducing the vision, mission, and core team (Me) to secure early interest and capital.

## Architecture
* **Frontend:** React/Vite (Onyxis-OS-UI)
* **Backend:** Python (LUPA-AI-Backend)
* **Cloud Platform:** Google Cloud Run (for Python/AI stability)
